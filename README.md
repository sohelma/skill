
# Project name

SkillSwap – A Local Skill Exchange Platform

## Purpose
An interactive platform for individuals to offer, learn, and trade skills within their local area. Whether it’s guitar lessons, language exchange, coding help,  yoga training and others — users can browse listings, rate experiences, and connect with local skill providers.

## Live URL


## Key features
- User authentication with Firebase (Email/Password and Google)
- Route-based SPA with protected Skill Details page
- Skill listings from a local JSON file
- Book session form with success toast
- User profile and update profile functionality
- Responsive design (mobile, tablet, desktop)
- Animations (AOS, Swiper slider) and toasts (react-hot-toast)

## npm packages installed
react-router, firebase, tailwindcss, react-icons, react-toast
